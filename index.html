복리마법

개인정보처리방침

시행일: 2026년 3월 16일 | 최종 수정일: 2026년 3월 16일

복리마법(이하 "서비스")은 이용자의 개인정보를 중요하게 생각하며, 개인정보 보호법 및 관련 법령을 준수합니다.

1. 수집하는 개인정보 항목

서비스는 아래 정보를 수집합니다.

  • 자산 입력 데이터 — 이용자가 직접 입력한 날짜별 자산 금액
  • 복리 설정 정보 — 목표 수익률, 수익 주기 설정값

📌 모든 데이터는 이용자의 기기 내 토스 네이티브 저장소에만 저장되며, 외부 서버로 전송되지 않습니다.

2. 개인정보 수집 및 이용 목적

  • 자산 데이터 저장 및 복리 계산 서비스 제공
  • 이전에 입력한 데이터 불러오기
  • 서비스 품질 개선

3. 개인정보 보유 및 이용 기간

이용자가 직접 삭제하거나 앱을 삭제할 때까지 기기 내에 보관됩니다. 서비스 운영자는 해당 데이터에 접근할 수 없습니다.

4. 개인정보 제3자 제공

서비스는 이용자의 개인정보를 제3자에게 제공하지 않습니다. 모든 데이터는 기기 내에서만 처리됩니다.

5. 외부 API 사용

서비스는 환율 정보 제공을 위해 아래 외부 API를 사용합니다.

  • Frankfurter API (api.frankfurter.app) — USD/KRW 환율 조회
  • 해당 API 호출 시 이용자의 개인정보는 전송되지 않습니다.

6. 투자 유의사항

⚠️ 본 서비스는 복리 수익을 시뮬레이션하는 계산기입니다. 실제 투자 결과와 다를 수 있으며, 투자에 대한 책임은 본인에게 있습니다. 과거 수익률이 미래 수익을 보장하지 않습니다.

7. 이용자의 권리

이용자는 언제든지 아래 권리를 행사할 수 있습니다.

  • 앱 내에서 입력한 데이터 직접 삭제
  • 앱 삭제를 통한 모든 데이터 완전 삭제

8. 개인정보 자동 수집 장치

서비스는 별도의 쿠키나 자동 수집 장치를 사용하지 않습니다.

9. 방침 변경

본 개인정보처리방침은 법령 및 방침 변경에 따라 내용이 추가, 삭제 및 수정될 수 있습니다. 변경 사항은 시행일 7일 전부터 서비스 내 공지합니다.
